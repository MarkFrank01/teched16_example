Welcome.

Thanks for your interest in my presentation UX202 - Building Services for Gateway. You can grab the ABAP code here and install it on your own system to look at it more closely and see it in action.

abapGit

To install the code onto you ABAP system you will need to use the abapGit project by @larshp.

If you haven't seen abapGit before I highly recommend you take a good look at it. If you find it useful perhaps you could consider joining the project and making your own contributions?

Setup

Before registering and activating the test Gateway services you will need to maintain the MPC and DPC locations. This is covered in the slides.

The relevant transactions are...

  • /IWBEP/R_DST_MODEL_BUILDER
  • /IWBEP/R_DST_SERVICE_BUILDER

For all other aspects about registering and activating SAP Gateway services consult the SAP documentation.

Enjoy!
